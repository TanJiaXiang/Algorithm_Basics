
# 题目

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。 n<=39 

# 解题思路

斐波那契数列有一个很重要的性质，后面一项是前两项的和(n>=3),前两项都是 1，很容易想到用递归求解第 n 项的值，但是递归需要消耗更多的内存，因此可以使用动态规划，用两个变量分别保持上次的计算结果和上上次的计算结果


```python
# 解法一：
class Solution: 
    def Fibonacci(self, n): 
        a = 0 
        b = 1 
        if n <= 0: 
            return 0 
        else: 
            for i in range(n): 
                a, b = b, a + b
            return a   
```
